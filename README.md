# ProjectBoost

https://garanceguinet.github.io/projectboost-idea-generator/

## Description

ProjectBoost est un générateur d'idées de projets web destiné aux développeurs débutants.
En un clic, l'utilisateur obtient une idée de projet à réaliser afin de s'entraîner en développement web.

## Fonctionnalités

- Génération aléatoire d'idées de projets web
- Empêche l'affichage de la même idée deux fois de suite
- Bouton désactivé temporairement pendant la génération
- Message de chargement simulé ("Génération en cours...")

## Technologie utilisées 

- HTML5
- CSS3
- JavaScript

## Comment utiliser le projet

1. Cloner le dépôt ou télécharger les fichiers
2. Ouvrir le fichier `index.html` dans un navigateur
3. Cliquer sur le bouton sur le boutton **Générer une idée de projet"**

## Objectif pédagogique

Ce projet a été réalisé dans un but pédagogique afin de pratiquer :

- La manipulation du DOM en JavaScript 
- La gestion des évenements ( `addEventListener` )
- L'utilisation des tableaux
- La génération de valeurs aléatoires
- La gestion d'un état (éviter les doublons)
- Les fonctions asynchrones avec `setTimeout`
